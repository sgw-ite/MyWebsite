html标准格式以及外部css和js的导入

html框架不赘述了（vscode万岁），需要注意的是每一个标签都要关闭，img要加入alt描述

外部css的导入的格式一般为： <link rel="stylesheet" type="text/css" media="screen" href="./css/css.css" />

js的导入：<script src="./js/hahaha.js"></script>

布局常用元素和方法

我自己在使用中布局主要还是使用div配合css布局，偶尔使用列表（主要是做导航栏）和span（拿来做类似markdown中的code框）

常用方法主要是相对路径+锚点的链接写法，‘<’和‘>’和‘ ’

导航栏：hover伪类的运用

我是使用列表来实现导航栏的。基本方法是：列表里面放链接，替换成块状元素，当鼠标指到链接时（hover）出发一系列举动，比如变色等。

下拉导航栏则是运用display：none和display：block的转换（也可以用visible：0和1来代替但是前者触发前是不占位的）

CSS渐变

（css渐变是我掌握的第一个是网页生动起来的效果^_^真心赞）

使用css渐变需要用类似transition:all 0.5s;的结构声明将会应用渐变的属性，然后指定时间（其实这和动画很相似）。

我个人常用于改变字体颜色和完成下拉菜单或着隐藏内容的渐入渐出

使目标在div中居中

1. margin: 0 auto;

这种方式常用于把div中指定元素居中（需要保证它占有一整行也就是下面的内容没有什么float之类的）

2. text-align: center;

这种可以是div中所有元素水平居中

3. display: table-cell; vertical-align: middle;

这种方法使元素垂直居中（常用于图片元素），但是要改变div种类

这个vertical-align属性应用的使用是相对与整行的，所以必须是用于行内元素，且如果不改成tablecell居中是没有反应的。。。（当然也可以手动将行高设置成div高，但若指定px那么不同比例下不同，这如果还要用js可太麻烦了= =）

4. display: box; box-align: center; box-pack: center;(注意：这里的box最好分开各种浏览器分别写)

这个用起来比较爽hhh就是IE完全不支持= =

CSS动画

我个人在这次的网页中很少用到动画，因为其实渐变就可以拿来做很多事。

在我的理解里，渐变很像一个定好起始和终点的动画，动画相较于渐变是拥有了更好的表达能力

keyframe格式和触发方法不赘述了，总之做的时候还是要记住注意各个浏览器的兼容问题

监听用户浏览位置

很多时候我们需要根据用户正在浏览的内容改变屏幕上某元素的状态（尤其是导航栏），这个时候就可以使用window.onsroll变量

具体代码的思路是将一个操作编写为函数，赋值给window.onscroll变量，由它来监听用户的滚动行为

这里又涉及到如何获取用户正在浏览的位置的问题，一般是通过body或者documentelement的scrolltop来获取（直接取个交集就完事了）

具体代码如下：

window.onscroll = function(e){
    var target1 = document.getElementById('xxx');
    var e =e || window.event;
    var scrolltop=document.documentElement.scrollTop||document.body.scrollTop;
    var windowheight = document.documentElement.clientHeight;
    if (scrolltop>windowheight) {
         pass;
     }
    if(scrolltop<=windowheight){
         pass;
    }
}

获取元素的绝对位置

很多时候这个要配合上面的函数使用

有的时候你希望屏幕滚到一个元素出现（或者是他消失）就进行一个什么动作，那么显然你需要获得他的位置。

如果像我一开始那样傻fufu的用px去试出元素的位置，不但很麻烦而且一旦你在别人的电脑上打开，你就会惊奇的发现：简直一团乱遭

所以就需要借助js来帮助我们计算元素的绝对位置

思路是用offsettop函数。

它的作用是输出元素到它的父级元素的顶端的位置，那么我们只需要逐级递归，直至父元素为document或者null为止，即可得到绝对位置

以下是具体代码：

function getabsolute(target) {
    var mytop = 0;
    var e = target;
    while(e != null){
         mytop += e.offsetTop;
         e = e.offsetParent
    }
    return mytop;
}
